# Lemonlime Linux ISO

This repository contains the tools, scripts, and configurations used to build the official Lemonlime Linux installation ISOs.

## About

Lemonlime Linux is a user-friendly, modern GNU/Linux distribution built from the ground up. This repository forms the foundation for how installation images are created, maintained, and released.

## Features

- Automated ISO build scripts
- Custom configurations for live media
- Support for producing test and release candidate images
