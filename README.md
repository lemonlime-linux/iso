# Lemonlime Linux ISO

Tools, scripts, and configurations for building Lemonlime Linux installation ISOs.

## About
This repository defines the ISO build process for the distribution, including live system configuration and build automation.

## Features
- Automated ISO creation  
- Custom live environment configuration  
- Support for test and release candidate images
